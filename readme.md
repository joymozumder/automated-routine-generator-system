laravel 5.7


## step 1:

```
git clone https://github.com/joymozumder/automated-routine-generator-system.git
```

## step 2: 

```
cd automated-routine-generator-system/
```

## step 3:

```
composer install
```

## step 4:

```
npm install
```

## step 5:

```
cp .env.example .env
```

## step 6:

```
php artisan key:generate
```

## step 7:

```
php artisan migrate --seed
```

## step 8:

```
php artisan serve
```

## step 9:

```
npm run watch
```
